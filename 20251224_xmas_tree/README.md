# Digital Xmas Tree 2025

2025年のクリスマスイブに、データサイエンスの力でツリーを構築しました。

## 技術的なこだわり
- **Geometry**: ツリーの葉は一様分布ではなく、$\sqrt{r}$ を用いたサンプリングを行うことで、中心から外側まで均一な密度になるよう調整しました。
- **Lighting**: `seaborn.hls_palette` を使用し、3D空間でどの角度から見ても美しく、かつ輝度が一定に保たれるような色彩設計を行っています。
- **Performance**: アニメーションフレーム内で `redraw=False` を指定し、ブラウザ側での描画負荷を最適化しています。

## 実行方法
`xmas_tree.ipynb` を実行すると、同ディレクトリに `index.html` が生成されます。
